### 存档文件结构

```python
分类 # 个人线/队伍线
  |
  --- 对象 # Kasssa / VKG
            |
            --- main.md # 事件记录文档
            |
            --- main.pdf # 事件记录文档的pdf版本
            |
            --- images # main.md需要的图片,存储的文件夹,可能没有
            |     |
            |     --- xxx.png
            |
            --- videos # main.md需要的视频,存储的文件夹,可能没有
                  |
                  --- xxx.flv
```
